# Password Strength Checker

## Domain
Cyber Security and Ethical Hacking

## Description
The **Password Strength Checker** is a simple cybersecurity tool designed to evaluate the strength of user passwords. It analyzes passwords based on key security parameters such as length, use of uppercase and lowercase letters, numbers, and special characters. The tool helps users understand whether their passwords are weak, medium, or strong, promoting better password hygiene and secure authentication practices.

## Objective
- To educate users about strong password creation  
- To reduce the risk of brute-force and dictionary attacks  
- To encourage secure cyber security practices  

## Features
- Checks password length  
- Detects uppercase and lowercase letters  
- Verifies numbers and special characters  
- Classifies password strength as **Weak**, **Medium**, or **Strong**  
- Simple command-line interface  
- Beginner-friendly Python implementation  

## Technologies Used
- Python 3  
- Regular Expressions (re module)

## How It Works
1. The user enters a password.
2. The program evaluates the password against predefined security rules.
3. Based on the evaluation score, the password strength is displayed.
